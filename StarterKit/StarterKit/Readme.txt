Component
========
1. Spread FarPoint 2.5 + Update patch
2. Oracle Client 10.2g , SQL Server 2000 or later
3. Crystal Report 11
4. EVOFramework data
5. EVOFramework windows form.

Step to initialize system
===============
1. Test database connection.
2. Test user authentication.
3. Load system maintenance (if need).
4. Load registered business module plugins.
    4.1 Search all file which matching with own pattern business.
    4.2 Load internal screen cache.
5. Load database screen cache.
6. Load image cache.
7. Startup menu form.
